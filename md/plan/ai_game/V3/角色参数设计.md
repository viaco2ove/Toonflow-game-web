# 需要从角色的描述等 通过ai获取到
1.获取后保存故事的角色表里的字段:parameterCardJson
2.游玩过程 绑定到存档相关的
目的存储游玩过程的参数值变化
## 游玩中的角色表里的字段（game_run_parameter）:parameterCardJson
游玩中的角色表：game_run_parameter
字段如下
id:一个故事的一个角色在在不同的玩家游玩产生不同游玩中的角色
game_run_id:每个玩家游玩时都会有一个游玩存档
parameter_id：角色id（包括用户这个特殊角色）
.... 角色的其他属性
parameterCardJson
