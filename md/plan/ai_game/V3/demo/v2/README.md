# Toonflow Demo v3

## 文件
- `index.html`：高保真可交互单页原型（Android + Web）
- `assets/`：参考图与素材

## 交互说明
- 顶部可切换 `Android 端` / `Web 端`
- 左侧切流程步骤
- 右上子状态按钮切换同一步的不同形态
- 首页右上角按钮：`进入故事大厅`、`语音开关（默认开启，点击后静音）`
- 首页推荐故事卡片点击：直接进入游玩故事
- 创建故事页：用户圆形头像右下角带编辑笔，点击可进入“用户扮演角色资料”编辑页
- 创建故事页：点击“新增”进入“新增角色资料”（头像/角色名/角色设定/音色/台词示例），点`确定`后自动切到编辑态并显示新角色
- 创建故事编辑态已补齐：创作指南、旁白面板、全局背景+提及、章节1（背景图/开场白/章节内容调试/提及/章节结局）、结局可见开关、背景音乐、添加下一章节、高级设定、下一步
- 游玩模式已补齐：实时 / 历史 / 故事设定 / AI提示（右侧悬浮按钮触发，3个可选回答 + 返回）
- `7`：在游玩对白上双击弹出菜单（复制/点赞/点踩/改写/重听），右上语音图标可开关自动播报
- `8`：支持“我的 → 设置 → 返回”页面切换

## 使用
直接打开 `index.html`。
