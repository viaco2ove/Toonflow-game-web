# "Toonflow Game" App Task1 Demo

## 文件说明
- `index.html`：原型入口页
- `web_prototype.html`：Web 全流程原型（与 Android 清单同业务链路）
- `android_prototype.html`：Android 全流程原型（按 `task1/android.md` 逐条映射）

## Android 原型覆盖
- 主页（随机推荐 + 输入框）
- 故事大厅（分类 + 搜索）
- 创建故事（角色、章节编辑、发布）
- 聊过（游玩记录）
- 我的与设置
- 游玩模式（单条/历史）
- 故事设定弹层
- 长按菜单、语音/文字输入切换、AI 回答建议

## 使用方式
直接在浏览器打开 `index.html`。
